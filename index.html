⚙️ One-time setup needed:

  1. Go to console.firebase.google.com
  2. Create a project → Enable Authentication → Google
  3. Enable Firestore Database (start in production mode)
  4. Go to Project Settings → copy the firebaseConfig
  5. Paste it into the FIREBASE_CONFIG object at the top of the <script>
  6. Add your hosting domain to Authentication → Authorized domains
